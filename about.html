About
This webpage was created as the final project for my mandatory course "The Basics of Web and Android Applications Development". As usual, every page uses HTML for the structure of the page and CSS to stylize it. Javascript was used to make the navigation bar stick on top of all content on every page. The 'Product' page heavily relies on Javascript as it uses a URL parameter to download product data and fit it on screen, as well as show the loading icon and notify the end user should an error fetching product data occur.
